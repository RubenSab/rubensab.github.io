About me and this site

I'm Ruben Sabatini, an undergraduate CS student from Rome.

I wanted a personal, independent space (a "sandbox") where I could safely collect my programming { projects } and a bunch of other things, like { interesting concepts I stumbled upon or tinkered with } and { photos }, away from the big and greedy social media platforms.

So I made this website that (now) I can truly call my own.

Before this one, I already had a "nice" website, but I mostly vibe-coded it with a LLM. Only now, while collecting the few useful scraps of the old site and designing this one, I'm realizing the massive untidiness and short-sightedness lying under the hood of the old one that I've been conditioned to accept in the name of convenience.

In short, make your own things yourself.
